This Web AI Application is by Gangesh Verma, a student of Bluebells School International. This Web Application is developed to support the mission of Swatch Bharat Abhiyan and promote proper waste disposal system in our everyday lives. We came forward with the idea of “Digital India” and use the camera of PC/Laptop/Tablet/Phone to detect if the waste is Biodegradable\Non-Biodegradable. India is a country with a population of 138 crores and so it is not practically possible to expect the government to take care of the cleanliness of the country. We all Indians need to aware before getting into action so that we can protect everybody's health and live in a hygienic environment. This is a demo of the Real Project. In this I have used a very pictorial library for waste detection. The more number of images we will have in the pictorial data, the better the accuracy of the detection of waste.

”Here all waste will be viewed as a recycling opportunity”
